# xr-storage背景说明

  • xr-storage 构想来源于开源库 good-storage；
  • 使用 typescipt 编写，拥抱下一代前端开发语言；
  • 使用rollup做library的构建；
  • 使用vuepress编写项目文档；
  • 使用jest编写了完整的单元测试。

# 能用xr-storage的优点？

  • 操作 localStorage 和 seesionStorage ，更加便利，开箱即用；
  • 内部实现了对象和字符串转换，无须手动将字符串转来又转去；
  • 浏览器禁用cookie不会报异常，只是会返回一个null；
  • xr-storage是一个混合 typescipt 对象，这意味着不仅可以当作一个方法，更可以使用它的属性。
